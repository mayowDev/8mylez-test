# 8mylez-code-test

- This repository is my solution to a code test provided by 8mylez, a potential company i want to work with.

- The technology i used in this project are HTML5 and CSS3. 

- The project is responsive on both mobile and tablet, as requested in the briefing. -> shorturl.at/bJOV7 .
- Full Template view:  https://www.portotheme.com/magento2/porto/demo3_en/

- To see the the Shopware version of the code please install the 'SwBasic.zip' directory in this repository as normal plugin in your shopware backend in the plugin manager. P.S: The theme is developed based on Shopware 5.6.7 version.

- Make sure to activate the plugin in order to see in theme manager section.

- Project demo link : https://shopware.netlify.app/
